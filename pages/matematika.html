Matemātika

Šajā lapā atradīsi trīs svarīgas tēmas, kas bieži parādās 9. klases matemātikas eksāmenā.

Pitagora teorēma

Formula

a²+b²=c²

  • a un b — taisnleņķa trijstūra katetes
  • c — hipotenūza (garākā mala, pretī taisnajam leņķim)

Piemērs

Ja a = 3 un b = 4, tad:

c2 = 3²+4² = 9 + 16 = 25 ⇒ c = √25​ = 5

Vizualizācija

Trigonometriskās attiecības (sin, cos, tan)

Formulas

sin(θ) = hipotenūza / pretējā katete​

cos(θ) = blakus katete / hipotenūza​​

tan(θ) = pretējā katete / blakus katete​​

Piemērs

Trijstūrī ar leņķi θ=30º, pretējā katete = 5, hipotenūza = 10

sin(30º) = 10 / 5​ = 0.5

Vizualizācija

Lineāru vienādojumu atrisināšana

Vispārīgā formula

ax + b = c​

Piemērs

2x + 3 = 11 ⇒ 2x = 11 -3 = 8 ⇒ x = 8 / 2 = 4
